## Overview

**ShopZone Backend** is a RESTful API built to serve the ShopZone eCommerce frontend. It handles user authentication, product catalog management, and order processing — all backed by a MongoDB database.

The API follows the **MVC (Model-View-Controller)** pattern, uses **JWT** for stateless authentication, and **bcrypt** for secure password hashing. It is fully deployed and connected to a live frontend.
